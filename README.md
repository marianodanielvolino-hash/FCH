# 🌿 Fundación Consciencia Humana — Landing Page

Landing page institucional de la **Fundación Consciencia Humana (FCH)**.

## ✨ Características

- **Diseño premium** con paleta de colores violeta, rosa, amarillo y verde
- **Tipografías**: Poppins (títulos) + Lato (textos)
- **Hero imagen** de amanecer inspirador a pantalla completa
- **Metodología tríada**: Lo Divino, La Matrix, Lo Terrenal
- **Métricas de impacto** con animaciones de conteo
- **Validación institucional**: GCBA y CAF
- **CTA de conversión** con WhatsApp y formulario de contacto
- **Diseño responsive** (desktop, tablet y móvil)
- **Animaciones suaves** con Intersection Observer

## 🚀 Deploy

Este sitio puede ser publicado directamente con **GitHub Pages**:

1. Ir a Settings → Pages
2. Seleccionar la rama `main` y la carpeta raíz (`/`)
3. Guardar — el sitio estará disponible en `https://<tu-usuario>.github.io/consciencia-humana/`
